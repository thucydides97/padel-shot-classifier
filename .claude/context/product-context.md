# Product Context

## Target Users

### Primary User: Padel Players
- Want to analyze their technique
- Interested in shot classification feedback
- May not have technical ML background

### Secondary User: Coaches
- Analyzing multiple players
- Need quantitative shot metrics
- Want visualization of technique differences

## Core Features

### Video Annotation Tool
- Manual shot boundary marking
- Shot type labeling (bandeja, vibora, smash)
- Frame-by-frame navigation

### Pose Extraction
- Automatic body keypoint detection
- Works with standard video formats
- Handles variable lighting conditions

### Shot Classification
- Three classification approaches
- Accuracy metrics and confusion matrices
- Feature importance analysis

### Visualization
- Shot comparison overlays
- Feature distributions
- Confusion matrices

## Use Cases

1. **Individual Practice Analysis**
   - Record practice session
   - Annotate shots
   - Get classification and technique insights

2. **Technique Comparison**
   - Compare feature distributions across shot types
   - Identify distinguishing characteristics
   - Improve weak areas

3. **Training Data Collection**
   - Build labeled dataset
   - Train custom classifiers
   - Improve accuracy over time

## Key Differentiators
- Uses biomechanical features, not just raw video
- Multiple classifier options
- Open source and customizable
